Set up devCanvas

devCanvas is not for serious use, but can be useful for lighter work using the canvas, like making objects collide. This article provides instructions on how to set up devCanvas for use in your own website(s).

Embedding devCanvas into a website is as simple as including the following markup in your website's source.

<body>
    ...canvas
    <script src="devCanvas.min.js">
</body>
